# NZ.CLI
> Цей клієнт робився в більшості своїй під мої потреби, тому доволі велику частину API я просто **НЕ ВИКОРИСТОВУЮ** (через те що вона мені не потрібна). Я зробив це задля того щоб можна було перевіряти оцінки та ДЗ не виходячи з терміналу.
 
 *P.S. В найближчий час я планую додати ще **TUI** та декілька інших **endpoints** (в основному для отримання сповіщеннь).*
## Що воно таке?
**NZ.CLI** - Мінімалістичний клієнт для **NZ.UA**, розробленний на **Go**. **Основна ціль**: *швидке отримання потрібних данних в короткому форматі*.

*Наразі підтримуються тільки деякі аргументи (в майбутньому їх стане трохи більше).*
## Аргументи для CLI
### Основні
Хочеться відразу зазначити, що для усіх цих параметрів треба вказувати аргументи `-start-date` та `-end-date` (інакше буде використовуватись тільки *сьогоднішня дата*).
- `-diary` - отримати домашнє завдання починаючи зі *start-date* і закінчуючи *end-date*
- `-perf` - отримати оцінки до кожного предмета та **правильну** семестрову на данний момент (тому що nz.ua рахує середню враховуючи абсолютно усі оцінки, а не тільки тематичні)
- `-grades` - отримати оцінки по предмету (зазначати в `-subject-id`)
### Додаткові
- `-subject-id` - **ID предмета в системі**. **NOTE**: Зараз я не знаю чи усюди він однаковий, тому не можу зробити більш легкий метод.
- `-start-date` - **Початкова дата**. Вказується у форматі `YYYY-MM-DD`.
- `-end-date` - **Кінцева дата**. Вказується у форматі `YYYY-MM-DD`.
*Усі аргументи можна отримати за допомогою параметра `-help`, бо ця документація може оновлюватись рідше аніж код*.
## Build
### Пререквізити
- Встановлений **Go 1.25+** (*рекомендовано*)
### Збірка
- **Клонувати репо** за допомогою `git clone https://github.com/TechAngle/nz-cli`.
- **Перейти** в директорію `cd nz-cli`.
- **Зібрати** за допомогою `go mod tidy && go build ./cmd/nz`
